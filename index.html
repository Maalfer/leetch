Leetch

Proxy MITM HTTP/HTTPS para pentesting web. Inspirado en Burp Suite.

Instalar en Debian / Ubuntu

# Añadir el repositorio (sin firma GPG)
echo "deb [trusted=yes] https://Maalfer.github.io/leetch stable main"   | sudo tee /etc/apt/sources.list.d/leetch.list

sudo apt update
sudo apt install leetch

Con firma GPG (si está configurada)

curl -fsSL https://Maalfer.github.io/leetch/leetch.gpg   | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/leetch.gpg

echo "deb https://Maalfer.github.io/leetch stable main"   | sudo tee /etc/apt/sources.list.d/leetch.list

sudo apt update
sudo apt install leetch
